kdu_client::is_reconnecting

Java: Kdu_client.Is_reconnecting

bool is_reconnecting( )

Java: public native boolean Is_reconnecting( )

Go to class description.

Synopsis

Returns true from the point at which reconnect is called to the point at which the first reply is received from the server, or close is called or disconnect completes.

This information can be useful, since target_started can only be used to determine the point at which the first reply is received from an original connect call, not from a reconnect call.
