kdu_cache::get_transferred_bytes

kdu_int64 get_transferred_bytes( int databin_class)

Go to class description.

Synopsis

This function returns the total number of bytes which have been transferred into the object within the indicated data-bin class. See add_to_databin for an explanation of data-bin classes. If we are attached to a different kdu_cache via attach_to, this function returns information for the cache to which we are attached.

Arguments

databin_class [int]
