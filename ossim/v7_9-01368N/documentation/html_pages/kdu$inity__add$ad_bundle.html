kdu_thread_entity_affinity::add_thread_bundle

void add_thread_bundle( int num_threads, kdu_int64 affinity_mask, kdu_int32 affinity_context)

Go to class description.

Synopsis

Adds a new thread bundle to the internal collection, identifying the number of threads in the bundle and the affinity_mask and affinity_context parameters to be passed to the kdu_thread::set_cpu_affinity function when the thread is created. For more information on these parameters, see the description of kdu_thread_entity::set_cpu_affinity.

If num_threads is non-positive, this function does nothing at all.

Arguments

num_threads [int]

affinity_mask [kdu_int64]

affinity_context [kdu_int32]
