kdu_cache_file_info::get_preamble_bytes

kdu_int32 get_preamble_bytes( )

Java: public native int Get_preamble_bytes( )

Go to class description.

Synopsis

A cache file may contain a preamble that consists of an initial subset of datab-bin descriptors, which play a particularly important role in summarizing the content. Typically, the preamble should be enough to render a reduced resolution (e.g., thumbnail) representation of the first image/frame/codestream in the source.

If there is a preamble, this function returns non-zero, and the returned value can be added to that returned by get_header_bytes to determine the total number of initial bytes from the cache file that are required to produce a usable representation of the source. Some applications might offer users the ability to truncate old cache files to just this minimum length for archival purposes.
