kdu_thread_entity_affinity

Java: class Kdu_thread_entity_affinity

[Declared in "../coresys/common/kdu_threads.h"]

[c++ namespace = "kdu_core::"]

Synopsis

This object exists to collect CPU affinity information ahead of a call to kdu_thread_entity::set_cpu_affinity. As explained with that function, the thread group owner and all working threads are partitioned into thread bundles, and each bundle can have its own affinity description. Bundles can be as small as one thread each, but normally the entire thread group would be described by only one or two bundles.
