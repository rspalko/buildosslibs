jp2_palette::get_abs_lut

Overload navigation: 1, 2

bool get_abs_lut( int lut_idx, kdu_sample32 lut[])

Go to class description.

Synopsis

This function is similar to the get_lut functions, except that it returns the relevant lookup table as absolute integers, rather than normalized quantities. If get_bit_depth indicates that the original palette entries had a precision B > 32, the original entries are downshifted by B-32, rounding down rather than to the nearest integer. Additionally, if get_signed indicates that the original palette entries were unsigned, they are first level adjusted by subtracting 2^{B-1}. Apart from these adjustments, the original palette entries are written as-is to the supplied lut array.

Arguments

lut_idx [int]

lut[] [kdu_sample32]
