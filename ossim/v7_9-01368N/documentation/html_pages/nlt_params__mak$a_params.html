nlt_params::make_gamma_params

Java: Nlt_params.Make_gamma_params

static void make_gamma_params( float p1, float p2, float gamma[])

Java: public native void Make_gamma_params( float p1, float p2, float[] gamma)

Go to class description.

Synopsis

Converts the p1 and p2 parameters of an NLTmake record into the E, S, T, A and B parameters (in that order) of the NLTgamma attribute, recording them in the gamma array, that is expected to have at least 5 parameters.

While this function is intended primarily for internal purposes, it can be used to construct the 5 parameters of an NLTgamma attribute from the two fundamental parameters p1 and p2, generating the others by enforcing continuity constraints. To assist with such usages, we note that:

  • p1 is the reciprocal exponent 1/E defined in connection with the NLTgamma attribute, which is usually known as the "gamma" values; common gamma values are 2.4 and 2.2, for example.
  • p2 is the offset parameter B defined in connection with the NLTgamma attribute. This is sometimes known as the "beta" values, which is generally close to zero. The sRGB gamma function, for example, has gamma=2.4 and beta=0.055.

Arguments

p1 [float]

gamma[] [float]
