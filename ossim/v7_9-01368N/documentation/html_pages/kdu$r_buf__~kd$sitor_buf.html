kdu_compositor_buf::~kdu_compositor_buf

Java: Kdu_compositor_buf.Native_destroy

virtual ~kdu_compositor_buf( )

Java: public native Native_destroy( )

Go to class description.

Synopsis

If you are providing your own buffers, based on a derived version of this class, be aware of the fact that the internal buf and float_buf arrays will be passed directly to the C++ delete operator if non-NULL. This is done for legacy reasons only. If you have done anything other than allocate the buffers using new (and you really should, because it is always preferable to at least align buffers on a 16-byte boundary), be sure to override the destructor and set them to NULL before exiting your version of the function.
