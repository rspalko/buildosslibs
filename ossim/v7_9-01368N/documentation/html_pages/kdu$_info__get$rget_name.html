kdu_cache_file_info::get_target_name

Java: Kdu_cache_file_info.Get_target_name

const char * get_target_name( )

Java: public native String Get_target_name( )

Go to class description.

Synopsis

Returns NULL if the object has not been initialized by a successful call to kdu_client::check_cache_file or has been reset since. Otherwise, the function returns string which has exactly the same interpretation as that returned by an active kdu_clients kdu_client::get_target_name function. In many cases this will be the same as the JPIP requested resource or target field, but it may include sub-target information and is not hex-hex encoded. This name is a useful consistent display name to use for the target that is represented by the cache file.
