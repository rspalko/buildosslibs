jp2_palette::get_abs_lut

Overload navigation: 1, 2

bool get_abs_lut( int lut_idx, kdu_sample16 lut[])

Go to class description.

Synopsis

Same as the first form of get_abs_lut, but writes the palette entries as 16-bit integers. If get_bit_depth idnicates that the original palette entries had a precision B > 16, the original entries are downshifted by B-16.

Arguments

lut_idx [int]

lut[] [kdu_sample16]
