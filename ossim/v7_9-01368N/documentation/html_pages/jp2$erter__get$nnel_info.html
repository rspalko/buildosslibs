Synopsis

This function is intended to help configure kdu_quality_limiter objects to take into account colour transforms that might be applied to a codestream's output image components. The function is invoked for each input channel ( idx) to discover: a) whether or not it is of chroma-type; and b) what scaling factor should be applied to the total squared error produced in that channel to determine the total squared error that will result across the transformed (sRGB) channels.

Return Value

False if idx does not correspond to a valid channel.

Arguments

idx [int]

square_weight [float &]
