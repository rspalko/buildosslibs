kdu_codestream::mem_failure

Java: Kdu_codestream.Mem_failure

void mem_failure( const char * mem_category, const char * message)

Java: public native void Mem_failure( String mem_category, String message)

Go to class description.

Synopsis

This function may be called if an internal error has occurred in connection with memory allocation. Currently, the function is called only if numerical overflow has occurred in a call to any of the pre-alloc functions offered by kdu_sample_allocator. In general, though, the function is called with a brief message that explains the failure (e.g., "internal error (numerical " "overflow)" and a mem_category string that identifies the sub-system that produced the error — neither argument should be NULL, as they are used to synthesize a suitable error message for kdu_error.

Arguments

mem_category [const char *]

message [const char *]
