kdu_simple_video_source::get_image_file_pos

Java: Kdu_simple_video_source.Get_image_file_pos

kdu_long get_image_file_pos( )

Java: public native long Get_image_file_pos( )

Go to class description.

Synopsis

Returns the absolute file position of the first byte in the currently open image's codestream. The behaviour of this function is undefined if open_image has not been called, or the last such call has already been matched by a call to close_image.
