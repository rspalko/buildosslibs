nlt_params::apply_rev_gamma

Go to class description.

Synopsis

See apply_fwd_gamma for an explanation. This function transforms values found on entry in lut according to the reverse gamma operator, mapping the sample values y that are recovered from decoding and inverse linear transform operators to output sample values z, which is the appropriate transformation for a decompressor.

num_entries [int]
