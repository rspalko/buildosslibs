kdu_cache_file_info::get_host_name

Java: Kdu_cache_file_info.Get_host_name

const char * get_host_name( )

Java: public native String Get_host_name( )

Go to class description.

Synopsis

Returns NULL if the object has not been initialized by a successful call to kdu_client::check_cache_file or has been reset since. Otherwise, the function returns a string which identifies the original host name (or IP address), including any port suffix (colon-separated) that may have been used.
