kdu_thread_entity_affinity::get_bundle_affinity

int get_bundle_affinity( int which, kdu_int64 & affinity_mask, kdu_int32 & affinity_context)

Go to class description.

Synopsis

Returns the 0 if which does not lie in the range 0 to B-1 where B is the number of thread bundles added via add_thread_bundle. Otherwise, returns the number of threads in the bundle identified by which and also returns the bundle's affinity parameters.

Arguments

which [int]

affinity_mask [kdu_int64 &]

affinity_context [kdu_int32 &]
