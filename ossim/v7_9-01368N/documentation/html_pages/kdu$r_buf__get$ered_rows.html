kdu_compositor_buf::get_rendered_rows

int get_rendered_rows( )

Java: public native int Get_rendered_rows( )

Go to class description.

Synopsis

Returns just the number of fully rendered rows out of the complete set of information returned via get_rendering_status. Rendering is complete once this value equals the height of the region returned by get_rendering_region.
