kdu_sample_allocator::finalize

Java: Kdu_sample_allocator.Finalize

void finalize( kdu_codestream codestream)

Java: public native void Finalize( Kdu_codestream codestream)

Go to class description.

Synopsis

Call this function after all pre-allocation (calls to pre_alloc and/or pre_alloc_block) has been completed. The function performs the actual allocation of heap memory, if necessary.

From KDU-7.3.1 onwards, this function requires you to pass in a non-empty codestream interface which is used to report any internal errors through codestream.mem_failure.

Arguments

codestream [kdu_codestream]
