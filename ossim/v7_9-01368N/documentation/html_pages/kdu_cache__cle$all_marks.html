kdu_cache::clear_all_marks

Java: Kdu_cache.Clear_all_marks

virtual void clear_all_marks( )

Java: public native void Clear_all_marks( )

Go to class description.

Synopsis

Returns all data-bins to the unmarked state, so that mark_databin will return 0. Note that this function does nothing if invoked on a kdu_cache object that has been attached to another (primary) cache via attach_to.
