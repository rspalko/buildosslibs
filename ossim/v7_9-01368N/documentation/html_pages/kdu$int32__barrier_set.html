kdu_interlocked_int32::barrier_set

void barrier_set( kdu_int32 val)

Go to class description.

Synopsis

Inserts a store memory barrier (store-fence) and then stores val in the relevant memory location. This ensures that all stores that logically precede this call will be globally visible by the time val becomes globally visible as the object's state value. This type of operation is also known as a "store with release semantics".

Arguments

val [kdu_int32]
