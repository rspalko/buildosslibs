kdu_compositor_buf::get_rendering_scale

Java: Kdu_compositor_buf.Get_rendering_scale

float get_rendering_scale( )

Java: public native float Get_rendering_scale( )

Go to class description.

Synopsis

Returns the scale at which this buffer has been (or is being) rendered. This is the product of the scale and rendering_scale_adjustment parameters that were originally passed in the relevant call to kdu_region_compositor::set_scale.
