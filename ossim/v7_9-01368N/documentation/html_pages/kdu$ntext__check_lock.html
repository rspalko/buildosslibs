kdu_thread_context::check_lock

bool check_lock( int lock_id, kdu_thread_entity * thrd)

Go to class description.

Synopsis

Returns true if thrd is currently holding the lock identified by lock_id — you should, of course, only invoke this function from the actual thread that is associated with thrd.

Arguments

lock_id [int]

thrd [kdu_thread_entity *]
