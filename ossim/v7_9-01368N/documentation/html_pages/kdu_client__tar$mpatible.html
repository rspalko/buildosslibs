kdu_client::target_incompatible

Java: Kdu_client.Target_incompatible

bool target_incompatible( )

Java: public native boolean Target_incompatible( )

Go to class description.

Synopsis

Returns true if the logical target-id returned by a JPIP server differs from the current value that was determined when the source was first opened or obtained from the original connect call. This is usually discovered after a reconnect call, whereupon any subsequent call to reconnect is bound to fail.
