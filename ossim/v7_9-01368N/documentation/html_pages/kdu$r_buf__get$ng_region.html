kdu_compositor_buf::get_rendering_region

Java: Kdu_compositor_buf.Get_rendering_region

kdu_dims get_rendering_region( )

Java: public native Kdu_dims Get_rendering_region( )

Go to class description.

Synopsis

Returns the dimensions and location of the region represented by this buffer within the overall composition whose location and dimensions are returned by get_total_composition_dims. The region returned by this function is expressed relative to the rendering scale (see get_rendering_scale) which depends on both the scale and rendering_scale_adjustment arguments that were passed to kdu_region_compositor::set_scale.
