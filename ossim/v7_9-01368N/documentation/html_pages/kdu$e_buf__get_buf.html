kdu_line_buf::get_buf

Java: Kdu_line_buf.Get_buf

void * get_buf( )

Java: public native long Get_buf( )

Go to class description.

Synopsis

Returns the same pointer as get_buf16 or get_buf32, whichever (if any) would have returned non-NULL. This function is provided only to avoid redundant conditional statements during certain generic operations that might work in the same way on all buffer types.
