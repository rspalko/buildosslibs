kdu_cache_file_info::get_header_bytes

Java: Kdu_cache_file_info.Get_header_bytes

kdu_int32 get_header_bytes( )

Java: public native int Get_header_bytes( )

Go to class description.

Synopsis

Returns the number of bytes in the header of the cache file (in practice, these bytes are all plain ASCII text). The header is immediately followed by a sequence of data-bin descriptors, each of which commences with a few bytes of binary identification information which are followed by the data-bin body bytes.
