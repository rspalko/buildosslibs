Synopsis

This function returns a relative value zeta that identifies the natural zero point for unsigned representations of the indicated colour channel. For most colour spaces the returned value is simply 0, and this also includes ICC colour spaces. For some enumerated spaces, however, the natural zero value of an unsigned representation is not 0. The return value is interpreted as a scaling factor on 2^B where B is the bit-depth of unsigned integers that might be used to represent the colour. Most enumerated colour spaces are formally defined with respect to a particular bit-depth (usually 8 or 10 bits/sample). When the channel is represented as an unsigned integer with this bit-depth the natural zero point Z is equal to zeta*2^B where zeta is the value returned by this function. Most opponent colour spaces have zeta=0.5 for the chrominance channels, meaning that unsigned 8-bit representations have an offset of 128 and offset 10-bit representations have an offset of 256. Some opponent spaces have luminance channels whose natural zero point is not exactly 0 either. For example, the JP2_YCbCr1_SPACE space has a luminance channel with Z=16 at 8 bit precision, so zeta = 16/256.

Knowledge of the natural unsigned zero point facilitates meaningful conversion between representations of different bit-depths and between signed and unsigned representations of each colour channel without any unnecessary loss of information.

Return Value

You can rely upon the fact that the zeta value returned by this function will never be smaller than 0.0 and never be larger than 0.75f; this range is sufficient to cover any reasonable colour space.

Arguments

channel_idx [int]
