kdu_cache_file_info

Java: class Kdu_cache_file_info

[Declared in "../apps/compressed_io/kdu_client.h"]

[c++ namespace = "kdu_supp::"]

Synopsis

This class is provided to encapsulate information retrieved via the static kdu_client::check_cache_file function, making it easier to obtain this information from both native and foreign languages such as Java or C#.

At the application level, you construct an instance of this of this class and pass it (by reference) to kdu_client::check_cache_file. If that function returns true, you can obtain meaningful information from the header of the cache file by querying the member functions provided here. After construction, pass the object to kdu_client::check_cache_file.
